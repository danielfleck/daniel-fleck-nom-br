Objetivo

Criei este site para reunir em um lugar anotações sobre as tecnologias que estudar. Com isso posso realizar consultas e ainda compartilhar este conteúdo com terceiros.

Como forma de aprendizado e de, futuramente, verificar a evolução do site, decidi iniciar ele do zero, tentando documentar todas as etapas de construção.

Etapas iniciais

  • 1. Registro de domínio
    • Fiz a aquisição do domínio diretamente do Registro.BR
    • Dentre as categorias de domínio, entendi que a mais compatível era .nom.br, para pessoas físicas
    • Essa categoria de domínio exige dois níveis, ou seja, no meu caso, daniel.fleck.nom.br.
  • 2. Criação de servidor na GCP
    • Para iniciar o projeto, criei no GCP uma instância de VM com as configurações que se enquadram nos Limites de uso do Nível gratuito: VM e2-micro não preemptiva, disco permanente padrão de 30 GB e um endereço IP externo.
    • Escolhi o SO GNU/Linux e a distribuição Ubuntu, em seguida realizei a atualização do SO e instalação do servidor web
  • 3. Configuração de servidor DNS
    • Eu decidi utilizar o Servidor de DNS do Registro.BR
    • A configuração é algo bastante simples, inclui inserir o nome de servidor e o ip externo fornecido pelo GCP
  • 4. Configuração de certificados TLS
    • Escolhi o Certificado TLS gratuito da Let's Encrypt
  • 5. Como primeiro missão, decidi criar uma interface para inserir conteúdo diretamente na página
    • Inicialmente, vou utilizar predominantemente a linguagem de programação PHP 8.2.0 e componentes do Symfony 6.2.2
    • Como sistema gerenciador de banco de dados vou utilizar o Postgresql na versão 15.1
Curitiba, 16 de dezembro de 2022.
